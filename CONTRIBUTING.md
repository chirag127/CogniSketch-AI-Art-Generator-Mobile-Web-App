# CONTRIBUTING GUIDELINES

## 1. Introduction
Thank you for your interest in contributing to CogniSketch-AI-Art-Generator-Cross-Platform-App! This project is dedicated to providing a high-quality, open-source AI art generator, and we welcome contributions from the community to help us achieve this goal.

## 2. Reporting Issues
If you encounter any issues with the project, please report them on the [Issues page](https://github.com/chirag127/CogniSketch-AI-Art-Generator-Cross-Platform-App/issues). When reporting an issue, please include as much detail as possible, such as:
* A clear description of the issue
* Steps to reproduce the issue
* Any error messages or logs

## 3. Contributing Code
To contribute code to the project, follow these steps:
1. Fork the repository: [https://github.com/chirag127/CogniSketch-AI-Art-Generator-Cross-Platform-App](https://github.com/chirag127/CogniSketch-AI-Art-Generator-Cross-Platform-App)
2. Create a new branch: `git checkout -b your-branch-name`
3. Make your changes and commit them: `git commit -m "Your commit message"`
4. Push your changes to your fork: `git push origin your-branch-name`
5. Open a pull request: [https://github.com/chirag127/CogniSketch-AI-Art-Generator-Cross-Platform-App/pulls](https://github.com/chirag127/CogniSketch-AI-Art-Generator-Cross-Platform-App/pulls)

## 4. Code Review
All pull requests will be reviewed by the project maintainers. We will check for:
* Code quality and readability
* Adherence to the project's coding standards
* Functionality and performance

## 5. Commit Messages
Please follow the standard commit message format:
`type(scope): brief description`
* `type`: One of `feat`, `fix`, `docs`, `style`, `refactor`, `perf`, `test`, or `build`
* `scope`: The scope of the commit (e.g., `ui`, `api`, `core`)
* `brief description`: A brief description of the changes made

## 7. Community Guidelines
We follow the standard open-source community guidelines:
* Be respectful and considerate of others
* Avoid spamming or self-promotion
* Keep the discussion focused on the project
